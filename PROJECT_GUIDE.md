# 🌱 دليل مشروع منظمة جودن للتنمية والسلام

## 📖 نظرة عامة

تم إنشاء موقع ويب كامل ومتجاوب لمنظمة جودن للتنمية والسلام باستخدام أحدث التقنيات:

- **Vue.js 3** (Composition API)
- **Vite** (نظام بناء سريع)
- **TailwindCSS** (تنسيق عصري)
- **Pinia** (إدارة الحالة)
- **Vue Router** (التنقل بين الصفحات)

---

## 🚀 البدء السريع

### 1️⃣ تثبيت التبعيات

```bash
npm install
```

### 2️⃣ تشغيل المشروع في وضع التطوير

```bash
npm run dev
```

سيتم فتح المشروع تلقائياً على: `http://localhost:3000`

### 3️⃣ بناء المشروع للإنتاج

```bash
npm run build
```

### 4️⃣ معاينة البناء

```bash
npm run preview
```

## 🧩 المكونات الرئيسية

### 1. Header (الهيدر)
- شفاف في البداية فوق صورة البانر
- يتحول إلى أزرق صلب عند التمرير
- ثابت (Sticky) في أعلى الصفحة
- متجاوب مع قائمة منسدلة للموبايل

### 2. Hero (البانر)
- صورة خلفية عالية الجودة
- تدرج لوني ناعم
- نصوص واضحة مع دعوة للفعل (CTA)
- إحصائيات اختيارية

### 3. Stats (الإحصائيات)
- عرض الإنجازات بالأرقام
- رسوم متحركة للأرقام عند الظهور
- أيقونات معبرة

### 4. ProjectsGrid (شبكة المشاريع)
- عرض المشاريع في شبكة متجاوبة
- فلترة حسب الفئة
- شريط تقدم لكل مشروع
- بطاقات قابلة للنقر

### 5. Gallery (معرض الصور)
- عرض الصور في شبكة جذابة
- Lightbox تفاعلي للعرض الكامل
- تنقل بالأسهم والكيبورد

### 6. SuccessStories (قصص النجاح)
- عرض دوار (Carousel) للقصص
- تشغيل تلقائي اختياري
- تنقل بالنقاط والأسهم

### 7. Forms (النماذج)
- نموذج التطوع
- نموذج التواصل
- نموذج التبرع
- التحقق من الإدخالات
- رسائل نجاح/خطأ

## 🔧 التخصيص

### تغيير الألوان

عدّل ملف `tailwind.config.js`:

```js
colors: {
  brand: '#0079BF',    // اللون الأساسي
  gold: '#F0B331',     // اللون الثانوي
  cream: '#FDF4E7',    // الخلفيات
  accent: '#CCE6F5'    // اللمسات
}
```

### إضافة مشروع جديد

عدّل ملف `src/stores/projects.js`:

```js
projects.value.push({
  id: 5,
  title: 'اسم المشروع',
  description: 'وصف المشروع',
  category: 'agriculture',
  // ... باقي الحقول
})
```

### إضافة صورة للمعرض

عدّل ملف `src/stores/media.js`:

```js
images.value.push({
  id: 7,
  title: 'عنوان الصورة',
  description: 'وصف الصورة',
  image: 'رابط الصورة',
  category: 'training',
  date: '2024-01-20'
})
```

## 🌐 النشر

### Netlify

1. ارفع المشروع إلى GitHub
2. اربط Netlify بالمستودع
3. إعدادات البناء:

1. ارفع المشروع إلى GitHub
